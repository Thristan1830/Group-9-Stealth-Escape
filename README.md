# Group-9-Stealth-Escape
Welcome to Stealth Pro!
 Stealth or fight the enemy to escape!
 Guns have limited ammo so use it wisely

Game is still in development, and has a lot of bugs.

Game controls:
W - MOVES FORWARD
A - MOVES TOWARDS LEFT
S - MOVE DOWNWARDS 
D - MOOVE TOWARDS RIGHT
Esc - Pause the game

left mouse button to shoot
The "R" Key: to reload

Your ammo is limited so conserve it.

Use angles when shooting and take cover when reloading



Enjoy!
